Project 1

Brain Game

In my project 1, i have created a Brain Game. In this game, User have to roll the
dice.If a number x appears then, he will have to play this quiz A and if y appears
he will have to play quiz B. However, the twist is that this game is based on points
system. If he scores 50, he will win the game however for each wrong answer,
points will decrease by 10. Furthermore, for each correct anser , user earns 10
points. However, if the points reaches below or equal to 10 , user will lose the game.
